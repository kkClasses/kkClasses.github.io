### Recent version oo Chrome or Mozilla (version 50+) may be used.
Infact you may update your browser -> Settings->help-> about browser

Open the file "recorder.html" with browser

There are following Options for recording
1. Camera/WebCam+ microphone/Mic
2. Any particular Window/PPT/ Brwoser Tab etc. + Microphone/Mic
3. Full Screen/Desktop + Camera/WebCam+ microphone/Mic


Once Recording starts, a popup window will appear at the top of the browser 
asking permission to access media.
